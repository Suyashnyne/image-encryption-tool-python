# Image Encryption & Decryption Tool

This project is a simple Python-based image encryption and decryption tool using pixel manipulation techniques.

## Features
- Encrypt images using a secret key
- Decrypt encrypted images
- Pixel-level manipulation
- Simple and beginner-friendly implementation

## Technologies Used
- Python
- Pillow Library

## Installation

Install Pillow library using:

```bash
pip install Pillow
```

## How to Run

```bash
python image_encryptor.py
```

## How It Works
The program modifies RGB pixel values using a numeric key to encrypt and decrypt the image.
